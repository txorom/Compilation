Le makefile se situe dans le dossier src

Pour compiler le compilateur il faut lancer la commande :
make

Pour exécuter les différents tests qui se trouve dans le dossier tst il faut lancer la commande :
make tst

Pour compiler le premier exemple il faut faire :
make ex

Pour le deuxième il faut faire :
make mandelbrot

Pour supprimer tous les fichiers inutiles :
make clean

Le fichier index.html ne se lance pas tout seul il se trouve dans le dossier src
